通过 shell 给 tmux 发送命令

14 September 2021
tmux new -t elastic -d 
tmux send -t elastic "~/elasticsearch-7.14.0/bin/elasticsearch" Enter

tmux split-window -t elastic
tmux send -t elastic "~/kibana-7.14.0-linux-x86_64/bin/kibana" Enter

新建会话 (可以使用 -d 直接后台进行创建)

tmux # 新建一个xx的会话
tmux new -s xx # 新建一个名称为xx的会话

切分会话窗口

tmux split-window -t xx

向当前会话窗口发送命令并执行

tmux send -t xx "$Command" Enter
